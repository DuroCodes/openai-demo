# OpenAI Demo Project

## Installation

**1.** Clone the repository.

```sh
git clone https://github.com/durocodes/openai-demo
```

**2.** Change directory into the repository.

```sh
cd openai-demo
```

**3.** Install the dependencies.

```sh
npm i
```

**4.** Add your API key to `config.ts`.

```ts
export const apiKey = 'your api key';
```

**5.** Run the project.

```sh
npm run start
```
